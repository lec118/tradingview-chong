# 자산 배분과 투자 심리

> 금융 생태계의 5대 자산(통화·주식·채권·부동산·상품)을 실전에서 관찰하고,
> 투자 심리의 **역발상** 매매 신호를 포착하는 스킬.

## 핵심 프레임워크: 5대 자산의 톱니바퀴 실전 적용

### 자산 간 관계 (2024~2026년 관찰)

```
금리 상승 → 달러 강세 → 주식 약세/채권 약세
금리 하락 → 달러 약세 → 주식 강세/채권 강세

[예외: 유동성 주입 시]
금리 동결 + SQE → 달러 약세 + 주식 강세 (비정상적 동행)
→ 이 예외가 바로 "유동성 장세"의 본질

[디커플링 현상 - 2025.09]
금리 인하 실시 → 장기채 금리 오히려 상승 (4.4%)
→ 재정적자 + 관세 인플레 기대가 장기 금리를 밀어올림
```

## 금(Gold)과 은(Silver) 분석

### 금 급등과 급락 — 2025.10의 교훈

| 시점 | 금 가격 | 변동 | 의미 |
|------|--------|------|------|
| 2025.10 고점 | **$4,400/oz** | 사상 최고 | 1979/2012 인플레 조정 고점 돌파 |
| 2025.10.21 | $4,123/oz | **-6.3% (하루)** | 2013년 이후 최대 일간 하락 |
| 은 동반 | — | **-8.7% (하루)** | 산업 수요 민감 자산 급락 |

### 급락 원인
- 장기채 금리 상승 → 달러 강세 → 금/은 매도
- 연준 금리 인하 지연 기대
- 인도 디왈리 축제 종료 → 현물 수요 소멸

### 시나리오 모델과 금 가격
- **1970년대 어게인 (스태그플레이션)**: 금 = 궁극의 수혜 자산 → 더 오를 수 있음
- **1980~90년대 어게인 (디플레)**: 금 = 하락 구간 → 주식으로 이동
- **현재 판단**: 상품의 대세 상승기 여부는 시나리오 모델에 따라 다름
- **금 선행성**: 통상 금이 먼저 대세 상승 → 이후 원자재 전체로 확산

### 금/은 비율 (Gold/Silver Ratio)
- 금/은 비율 80 이상 → 경기 불안 극대화
- 금/은 비율 60 이하 → 경기 호황
- **실전**: 금/은 비율이 정점을 찍고 하락하면 → 경기 회복 신호

## 비트코인 분석

### 비트코인 = 유동성의 가장 순수한 바로미터
- 비트코인 폭락 = 유동성 축소의 **첫 번째 신호**
- 비트코인과 나스닥 상관계수 높음 → **비트코인이 1~4개월 선행**
- **역사적 패턴**: 비트코인 정점 → 1~4개월 후 → 주식시장 정점

### 2025~2026년 핵심 데이터
| 시기 | 가격 | 이벤트 |
|------|------|--------|
| 2025.10.21 | — | **$191.5억 역대 최대 마진콜** (27조원) |
| 2025.11 | $102K | $100K 심리선 이탈, $108K 수평선 저항 |
| 2025.11 | $110K→$102K | 정점 후 하락 |
| 2026 | $65K~ | $100K→$65K 하락 진행 중 |

### 마진콜 원인 분석 (2025.10.21)
- 중국 희토류 수출 제한 + 트럼프 100% 관세 위협
- **롱/숏 비율 6.7:1** → 극단적 레버리지 편중
- 주말 저유동성 + 연쇄 청산 → 폭포식 하락
- 2008년 서브프라임과 심리 구조 유사 (그리드 → 패닉)

### 비트코인 가격 수준별 해석
| 가격대 | 의미 |
|--------|------|
| 100,000불+ | 유동성 과잉, 투기 과열 |
| 80,000~100,000불 | 정상 범위 (기관 참여 구간) |
| 65,000~80,000불 | ⚠️ 유동성 축소 시작 — 현재 |
| 50,000불 | 2021년부터의 장기 횡보 지지선 |
| 40,000불 이하 | 심각한 유동성 위기 → 주식 중공황급 |

## 달러 방향성 — 제3차 달러 약세 시작

### 달러의 역사적 3대 약세기
```
1차 약세: 1971년 닉슨 쇼크 → 금 태환 중지 → 10년 약세
2차 약세: 1985년 플라자 합의 → 달러 의도적 절하 → 10년 약세
3차 약세: 2025년~ → 재정 적자 + 금융 억압 → 진행 중 ← 현재
```

### 제3차 달러 약세의 원인
1. **금융 억압(Financial Repression)**: 인플레 속 정치적 압력으로 금리 인하 → 달러 실질가치 하락
2. **재정 적자 확대**: 부채한도 $5조 상향 → 국채 대량 발행 → 달러 가치 희석
3. **관세 딜레마**: 관세 → 수입물가↑ → 인플레↑ → 그런데 달러는 약세
4. **글로벌 탈달러**: 중국/러시아의 위안화·금 결제 확대

### 달러 약세의 투자 함의
- **비미국 자산(EM/유럽) 강세**: 달러 약세기 = 신흥시장 초과 상승기
- **한국 수혜**: 원화 강세 환경 → 외국인 자금 유입 → 코스피 상승
- **원자재 강세**: 달러 약세 → 원자재 가격 상승 → 상품 사이클 시작

### 달러 강세 요인 (단기 반등 가능)
- 엔화/파운드 약세에 의한 **상대적 강세** (2025.11)
- 연준 금리 인하 유보 → 금리차 유지
- 미국 정부 셧다운 → 세이프 헤이븐 수요
- 단기 신용경색 → 달러 캐시 수요

## 채권 전략

### 2020년대 — 역대 최악의 채권 수익률
- 10년물 실질 수익률: **역사적 최저** 수준
- 장기채 실질 수익률: **-8.3%** (20년대 누적)
- 2022년 장기채 **-40% 폭락** → 안전자산 지위 상실 신호
- **60/40 포트폴리오 실패**: 채권이 더 이상 주식의 헷지 역할 불가

### 인플레이션 시대 전환 (2020~2050)
- 1980~2020: 저인플레/저금리 시대 → 채권 황금기
- **2020~2050: 인플레이션 시대** → 채권 장기 약세
- **TIPS(물가연동채)**: 구조적 인플레 시대의 핵심 헷지 수단

### 채권 포지션 전환 시기
- scenario_investing의 **겨울 국면** 진입 시 → 달러 표시 단기 채권으로 전환
- 현재(가을 국면) → 주식 70~80% + 채권/현금 20~30%
- 장기채 매수는 **금리 정점 완전 확인 후에만**

## 투자 심리 분석 (역발상의 실전)

### 월가 비관론자의 항복
- **의미**: 약세론자들이 전향하는 시점 → **정점 임박**
- 2025.10: 모건스탠리 반도체 하락론 → **'A Warm Winter'** 상승론 전환
- 월가 약세론자 상당수 전향 → ⚠️ 경계 신호

### 개인투자자의 역발상 행동
- 2025.10.04: 코스피 역신가 돌파 날 **개인 2조원 하루 투매** → 전형적 소외 매매
- 외국인이 3.87조 매수할 때 개인은 공포 매도 → **역발상 매수 신호**

### 탐욕-공포 지수 활용
| 수준 | 상태 | 대응 |
|------|------|------|
| 0~25 | Extreme Fear | 역발상 매수 검토 |
| 25~50 | Fear | 관망 |
| 50~75 | Greed | 정상 보유 |
| 75~100 | Extreme Greed | ⚠️ 비중 축소 검토 |
