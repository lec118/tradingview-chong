# 공황 시나리오와 위기 대응

> scenario_investing의 `04_ground_zero`와 `11_kitchin_cycle`의 **실전 적용**.
> 현재 시장에서 발생 가능한 공황 시나리오를 구체적으로 분석한다.

## 현재 시점의 공황 시나리오 (2026년 기준)

### 시나리오 A: 소공황 (가능성 높음, 60%)
- **트리거**: 금리 동결 장기화 + TGA 증가에 의한 유동성 흡수
- **규모**: S&P 500 기준 **10~15% 조정**, 200일 이내 회복
- **패턴**: 키친 사이클 3~4년 주기에 의한 순환적 조정
- **대응**: 주도 업종(반도체) 보유 유지, 현금 비중 20~30%
- **참고**: 2025.09 소공황 이미 발생 후 반등 (금리 인하 5개월 후)

### 시나리오 B: 중공황 (가능성 중간, 30%)
- **트리거**: 인플레이션 재점화 + 금리 재인상 + 신용경색 확산
- **규모**: S&P 500 기준 **30~50% 하락**, 2~3년 약세장
- **패턴**: 2000년 닷컴, 2008년 서브프라임과 유사
- **전제 조건**: 3대 금융위기 선행조건 충족 (충분한 버블 + 충분한 시간 + 충분한 전조현상)
- **시기**: 인플레 재개 + 금리 인상 전환 시 → **2027~2028년** 예상
- **대응**: 주식 전량 매도 → 달러/채권 전환

### 시나리오 C: 연착륙→재상승 (가능성 낮음, 10%)
- **트리거**: 연준의 완벽한 금리 조율 + AI 혁명의 실질 수익화
- **패턴**: 1995~1996년 연착륙과 유사
- **대응**: 주도 업종 적극 보유

## 핵심 판단 프레임: 금리 인하와 공황의 관계

### ⚠️ 역사적 철칙: 4번 이상 금리 인하 → 항상 공황

| 인하 횟수 | 결과 | 예외 |
|---------|------|------|
| **4번 이상** | **항상 강한 공황(BC) 발생** | **역사적 예외 없음** |
| **3번 이하** | 거의 항상 소공황(FC) | 유일한 예외: 1995~96 |

- 현재(2026.03): 2번 인하 완료 → **4번 넘어가는지 모니터링 필수**
- 금리 인하 시작 후 **5~6개월 시차**로 공황 패턴 발생 (역사적)
- 수익률 곡선 역전 해소(역전→정상화) 시점 = **1급 공황 전조현상**

### 금리 인하 = "좋은 징조"가 아니다 (1급 전조현상)
- 대중적 인식: 금리 인하 → 경기 부양 → 주가 상승
- **이야기꾼 관점**: 금리 인하 → 이미 경기 하강 진행 → 뒤늦은 대응
- 역사적으로 금리 인하 이후 **자산 폭락은 시차를 두고 발생**

## 스태그플레이션 — 숨겨진 최대 리스크

### 2025.09 스태그플레이션 초기 국면 진단
- CPI 2.9% → 하락 추세 **멈추고 반등 조짐**
- 고용 데이터 약화 (구인 감소, 실업률 상승) + 경기 성장 둔화
- 물가 ↑ + 경기 ↓ = **스태그플레이션 초기**
- 연준 딜레마: 인플레 미진압 + 경기 부양 필요 → 가장 어려운 정책 환경

### 인플레이션 재점화 경로
```
CPI yoy < 2.5% → 안전 (금리 인하 환경)
CPI yoy 2.5~3.5% → 주의 (금리 동결 연장) ← 현재 근처
CPI yoy > 3.5% → 위험 (금리 재인상 가능성)
CPI yoy > 4.0% → 중공황 시나리오 본격 작동
```

## 자산 폭락 순서 패턴 (역사적 관찰)

```
1. 투기 자산 하락 시작 (비트코인, 밈스톡, SPACs)
2. 귀금속 급등 후 하락 (금/은 - 안전자산 수요 피크)
3. 대장주 약세 전환 (엔비디아, 테슬라)
4. 지수 하락 (S&P, 나스닥)
5. 신용 경색 확산 (회사채 스프레드 급등)
6. 공포 극대화 (VIX 40+)
```

### 2026년 현재 상태
- **1단계 진행 중**: 가상화폐 $191.5억 역대 최대 마진콜 (2025.10.21), 비트코인 $102k→$65k↓
- **2단계 일부 관찰**: 금 $4,400 사상 최고 → -6.3% 하루 급락 (2013년 이후 최대)
- **3단계 일부 관찰**: 애플, 구글 하락 → 엔비디아는 아직 $5T+ 강세
- **4~6단계**: 미발생

## 셧다운 vs 부채한도 — 시장 영향 구분

| 구분 | 정의 | 시장 영향 |
|------|------|---------|
| **셧다운** | 예산안 미승인 (돈 있지만 못 씀) | 21회 중 **14번 상승** → 일시적 노이즈 |
| **부채한도** | 빌릴 권한 없음 | 심각 → 최우선 해소 대상 |

- 2025.10: OBBB 법안으로 부채한도 $5조 상향 → 여유 확보
- 셧다운은 **대세 상승장을 훼손하지 않음** (역사적 패턴)
- 단, 경제 데이터 발표 지연 → 연준 불확실성 일시 증가

## 위기 대응 매뉴얼

### 소공황 대응 (200일 이내 회복 예상)
1. 현금 비중 20~30%로 확대
2. 주도 업종(반도체) **코어 포지션 유지**
3. 레버리지 포지션 즉시 정리
4. 하락 10% 시점에서 분할 매수 시작

### 중공황 대응 (2~3년 약세장)
1. 모든 주식 포지션 정리
2. 달러 표시 채권/단기 국채로 전환
3. 매매 중단 → **scenario_investing의 겨울 국면 대응**
4. 다음 봄 국면의 전조현상이 나올 때까지 대기
