# 메모리 반도체 캐퍼 증가는 내년 중반 이후

| 항목 | 전망 |

| 주가 정점 (실적 1년 선행) | **2027년 중반 이후** |

## 메모리 가격 동향

- InSpectrum 자료 기준: DDR5 DRAM 현물가와 NAND Flash 512Gb 현물가가 **2025년 3분기부터 급등**
- DDR5 DRAM 현물가: 약 600% 수준까지 상승 (2024년 중반 대비)
- 가격 급등과 쇼티지가 가중되면서 캐퍼 증설이 불가피한 상황

## 기업별 증설 계획 상세

### 마이크론 (Micron)
- **본사**: 아이다호주 보이시에 **500억 달러** 투자
  - 캠퍼스 면적 2배 확장
  - **팹 2개** 추가 건설
  - 첫번째 팹: **HBM 전용 라인**, 2027년 중반 출하 계획
  - 두번째 팹: 2028년 말 첫 출하 전망
- 뉴욕 시라큐스: **1천억 달러** 투자 신규 팹 착공
- 일본 히로시마: 추가 투자해 증설 발표
- 이들은 **2028년에 첫 출하** 시작

### 삼성전자
#### P4 (평택)
- 3~4nm 이하 첨단 공정 대응
- **P4-Ph4**: 클린룸 장비 FFU(Fan Filter Unit) 올해 **8월** 유틸리티 가동 시작
  - 장비 반입: 올해 하반기
  - 수율 작업 감안 → **내년 상반기쯤 (5월??)** 공급 기여 가능
  - 클린룸 가동 이후 3분기 이후로 보면 됨
- **P4-Ph2**: 올해 **12월** FFU 유틸리티 가동 → 2027년 초 장비 반입/세팅
  - 대충 클린룸 가동 시작하고 **3분기 정도** → 2027년 4분기에 공급 기여
- 내년 중후반부터 수율 측면에서 의미 있는 물량 공급 시작 전망
- **26년 90K → 삼성전자의 일반 DRAM MS 확대** 가능

#### P5 (평택)
- **Pulling 검토 중** → P5 fab open **2028년 하반기~2029년**
- 2028년 중반이 되어야 공급 증가에 기여 가능

### SK하이닉스
#### M15X (청주)
- Ph3: DRAM 40K → **Pulling 검토 중**
- Ph4: DRAM 40K → **HBM** 전환
- 공급 기여 시점: 2027년~

#### Y1 (용인)
- 일정 단축 해봐야 **2028년 초**, Y2도 2028년 중반 이후 공급 증가
- **Y1**: 2027년 2월에 FFU 유틸리티 가동하도록 일정을 **4개월 단축** 예정
  - → 2028년 초나 되야 공급 기여 가능
- **Y2**: 2027년 8월에 FFU 유틸리티 가동하도록 일정 무려 **6개월 단축** 촉할 예정
  - → 2028년 중반쯤 공급 증가 기여
- **Y1 fab open: 2027년 2~5월**

### 마이크론 (Fab16/ID1)
- **Fab16 (A3)**: Ph2: DRAM 15K → Ph3: DRAM 15K → **HBM**
- **ID1**: **ID1 fab open 1H27** (2027년 상반기)

## CSP 4사 vs 메모리 3사 CAPEX 비교

> 자료: Bloomberg, TrendForce, SK증권

- CSP 4사 (Amazon, Google, Meta, MS)의 CAPEX가 2024~2025년 급증
- 이것이 **메모리 병목 유발** → 메모리 CAPEX 확대 **필연적 후속**
- 2026E~2028E: 메모리 3사 CAPEX 증가세 가속 전망
- **핵심 인사이트**: CSP CAPEX 증가 → 메모리 수요 폭발 → 메모리 CAPEX 증가 → 2~3년 후 공급 과잉 → 사이클 정점

## 실적 전망

- 올해 삼성전자와 SK하이닉스의 예상 매출액 증가율: 각각 **192%, 213%**
- 실적 증가세가 **관성으로만 움직여도 내년까지 지속** 전망
- 코스피 12개월 예상 순이익(좌축)과 DRAM Exchange Index(우축)의 상관관계 높음

## DRAM Exchange Index와 코스피

- **DRAM Exchange Index 정점이 대부분 공황 직전일 가능성 크다** → 이번에도 주목
- 도표 9: DRAM 가격 상승이 지속될지 여부와 코스피 12개월 예상 순이익은 거의 동행
- 도표 10: **반도체를 제외한 코스피는 3,620선** → 반도체가 코스피의 핵심 드라이버

## 투자 시사점

1. **캐퍼 증가 본격화까지 시간적 여유**: 2027년 중반까지 공급 부족 지속
2. **주가 정점은 실적보다 1년 선행**: 실적 정점 2028년이면 → 주가 정점 2027년 중반 이후
3. **현재는 저평가 구간**: PER 11배 미만, EPS 증가 속도 > 주가 상승 속도
4. **DRAM Exchange Index 모니터링 필수**: 정점 시 공황 직전 가능성
5. **반도체 제외 코스피 3,620 → 반도체 의존도 매우 높음**: 반도체 약세 전환 시 코스피 충격 클 것
