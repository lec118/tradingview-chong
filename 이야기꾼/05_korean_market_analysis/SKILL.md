# 한국 증시와 수출 사이클

> scenario_investing의 `06_korean_market`가 이론적 프레임이라면,
> 이 스킬은 한국 증시 **4기 봄 국면의 실전 분석**이다.

## 현재 위치: 한국 증시 4기 봄 국면 (2025~)

> ⚠️ 기존 스킬에서 '3기 가을'로 기재했으나, 83개 포스트 정밀 분석 결과
> 이야기꾼은 **4기 봄 국면 진입**으로 판단을 전환했음.

### 코스피 4기 개요 — 18년 더블링 사이클
```
코스피 1기: 1987년~  (100p → 1,000p)
코스피 2기: 1999년~  (300p → 2,000p) 
코스피 3기: 2005년~  (1,000p → 2,700p)
코스피 4기: 2025년~  (2,000p → 4,000p~5,000p?) ← 현재
```

### 2025~2026년 시장 상황
- 코스피: **3,549p 역사적 신고가** 경신 (2025.10.04)
- 외국인 주간 3.87조원 '폭탄 매수' (삼성전자 하루 1.7조원)
- 개인 역대 최대급 3.07조원 매도 → 포모 현상 대기 자금 축적
- PER 10~11배 → 글로벌 대비 **극도의 저평가** 지속
- 코스피 이익 2024~2026년 **3년 연속 성장**, 2025년 **역대 최대** 전망

### 코스피 5,000 가능성 분석
- **일본 벤치마크**: 2014년 이토 리포트 → ROE 8% 기준 강화 → PBR 상승
- **대만 벤치마크**: 1998년부터 주주 환원 강화 → 평균 배당 55.3%, ROE 15.3%
- 대만 PBR 2.4x vs 한국 PBR 0.9x → 코스피 밸류업 성공 시 **구조적 재평가 잠재력 극대**
- **⚠️ 경고**: 2026년 이익 급증(261조원, +26%) → **2027년 고주의** (이익 급증 이듬해 조정 패턴)

## 핵심 프레임워크: 수출 사이클 정점 신호

### 수출 증가율과 주가의 관계 — 가장 중요한 신호
한국 경제의 특성상 **수출 데이터**가 증시의 가장 빠른 선행지표:

```
수출 yoy 바닥 → 6~12개월 후 → 코스피 바닥
수출 yoy 정점 → 6~12개월 후 → 코스피 정점
```

### 수출 실적 핵심 데이터
| 시점 | 수출액 | yoy | 비고 |
|------|--------|-----|------|
| 2025.08 | $584억 | +1.3% | 증가세 **둔화 시작** — 경계 |
| 2025.09 | $659.5억 | +12.9% | **3년 6개월 만에 역대 최대** |
| 2025.10 | $595.7억 | +3.6% | 10월 기준 역대 최대 |
| 2026.01 | — | +34% | 반도체 주도 급증 |

- **반도체 수출**: 2025.09 $141억(+38.8%), 2025.10 $157.3억(+25.4%)
- 대만 수출 +46% 급증 (HBM 공급)
- 조선·화장품·라면·전기기기 등 일제히 9월 역대 최대
- EU(+19.3%), 중남미(+34%), 아세안(+17.8%) 대부분 역대 최대

### 수출 경고 신호
| 신호 | 상태 | 해석 |
|------|------|------|
| 수출 yoy 30% 초과 | ⚠️ **2026.01 현재 34%** | 사이클 후반부 진입 |
| 반도체 수출 yoy 100% 초과 | ⚠️ **2026.01 현재 102.7%** | 과열 구간 |
| 수출 증가율 yoy 3개월 연속 둔화 | 미발생 | 이 신호가 나오면 정점 확인 |
| 자동차 수출 마이너스 전환 | ⚠️ **2025.10 -16.2%** | 트럼프 관세 영향 시작 |
| 8/15 수출 품목 감소 | ⚠️ **2025.08 발생** | 반도체·자동차·조선 외 모두 감소 |

### 경기 과열 경고
- 제조업 지수 급격한 회복 + 수출 기록 경신
- 2026년 **내수 소비까지 회복되면 경기 과열(Overheating) 구간 진입** 가능
- 과열 → 인플레 → 금리 인상 전환 → 중공황 시나리오 연결

## 코스피 4기 봄 국면의 특성

### 저PBR 장세 (1990년 어게인)
- 보험·은행·증권 등 저PBR 업종 상승 → 정부 밸류업 정책 효과
- 자사주 소각 의무화 → PBR 구조적 재평가
- **⚠️ 밸류에이션 앵커링 편향 경고**: PBR이 과거 평균보다 높다고 매도하면 안됨 → 구조적 가치 변화 중

### 밸류업 프로그램
- **긍정적 효과**: 저PBR 금융주, 지주회사 재평가 → 이미 2024~2025년 1차 효과 반영
- **한계**: 일본 밸류업과 달리 **지배구조 개혁 실효성** 의문
- **실전 적용**: 밸류업 테마는 **지속성보다 이벤트 드리븐**으로 접근

## 환율과 코스피의 관계

### 핵심 원칙 (scenario_investing 이론 적용)
- **원화 강세(환율 하락)** → 코스피 상승에 유리
- **원화 약세(환율 상승)** → 코스피 하락 압력

### 실전 판단
```
원달러 환율 1,300원 이하 → 대세 상승 환경
원달러 환율 1,300~1,400원 → 중립~약세
원달러 환율 1,400원 이상 → 위험 신호 → 외국인 자금 유출 우려
원달러 환율 1,500원 → 구조적 변화 (2025.11 분석)
```

### 한미 금리차 역전의 의미 (scenario_investing `08_precursors` 보완)
- 한미 금리차 역전 = 한국 금리 < 미국 금리
- 뉴스에서는 "자본 유출 우려"라고 하지만 실제로는 **대세 상승 초기 신호**
- 원화 강세 자리이므로 환차익 + 시세 차익 가능 → 오히려 자본 유입

## 코스피 업종별 순환 (한국 시장 특성)

### 4기 주도 업종 변천
```
2020~2021: 2차전지, 인터넷/게임, 바이오
2022~2023: 조정기 → 전 업종 하락
2024~: 반도체(HBM/AI), 조선, 방산
2025~: 저PBR(보험·은행·증권) + 반도체 지속
향후: 소비재? (중국 이구환신 수혜 가능)
```

### R&D 투자 — 정부의 기술 산업 지원
- 2026년 R&D 예산 **35.3조원** (+19.3% YoY) → 10년 최고 성장률
- AI 전용 R&D 예산 **2.3조원** → 전년 대비 **+109%**
- 국가성장펀드 150조원 목표 (기존 100조원에서 확대)
