## 현재 위치: 미국 증시 7기 가을 국면 (2024~)

### 80년 시나리오 모델에서의 위치
- 미국 증시 8번째 마스터 사이클의 **후반부 (Summer/Autumn)**
- 미국 → EM(신흥시장) 로테이션 구간 시작
- **"죽음의 선" 기술적 분석** vs **시나리오 모델**: 기술적 지표만으로 -65% 폭락을 예측하는 것은 위험 → 80년 사이클 기반의 시나리오 모델이 더 신뢰성 높음

### 2025~2026년 시장 상황
- S&P 500: **6,791p** 사상 최고 (2025.10.26)
- 나스닥: 7,000p 돌파 후 6,800~7,000p 구간 박스권 횡보
- 불마켓 3년차 → 과거 패턴상 **깊지 않지만 지루한 조정** 국면
- 지수는 버티지만 개별 종목 분화 심화 (Breadth 약화)
- **엘리어트 파동 분석**: 이번 불 사이클 최종 정점 **7,200p** 가능

## 핵심 분석 프레임

### 1. 박스권 분석 — 에너지 응축 모델 (공수교대 수평선)
- **공수교대(攻守交代)**: 한 번 돌파한 저항선이 지지선으로 전환되는 현상
- S&P 500 6,750p: 10월 중순 저항선 → 돌파 후 **지지선으로 전환** (2025.11)
- 박스권이 길수록 → 이탈 시 움직임이 더 크다
- 상향 돌파 실패 2~3회 → 하방 돌파 가능성 급상승

### 실전 판단
```
나스닥 24,500p 하향 돌파 → 약 1년간 긴 조정
나스닥 21,000p 상향 돌파 → 가을 국면 마지막 상승파
S&P 500 6,200p → 200일 이평 근처, 깊은 조정 시 지지
```

### 2. 천정권 다이버전스 — 가장 중요한 위험 신호
- **정의**: 지수는 고점 근처이지만, 상승 종목 수가 감소하는 현상
- **측정**: NYSE Advance-Decline Line, % of stocks above 200MA
- **해석**: 소수 대형주가 지수를 떠받치는 구조 → 불안정
- 2025~2026년: 엔비디아 제외 시 Mag7 대다수 하락 추세 → 전형적 다이버전스
- **별리 종목 수가 감소하기 시작하면**: 지수 신고가와 무관하게 **정점 임박**

### 3. VIX와 풋옵션 분석
- **VIX 20 이하**: 시장 안일 (탐욕) → 갑작스런 조정 가능성
- **VIX 20~30**: 불안 증가 → 조정 진행 중
- **VIX 30+**: 공포 극대화 → 역발상 매수 기회 탐색
- **풋/콜 비율 0.67**: 역사적 위험 구간 근접 (2025.11 관찰)
- **MOVE 지수** (채권 VIX): 안정화 중 → 채권 시장 리스크 완화 신호 (2025.11)

### 4. 불마켓 연차별 특성
| 연차 | 특성 | 조정 패턴 |
|------|------|---------| 
| 1년차 | 바닥에서 급등 | 깊은 조정 없음 |
| 2년차 | 안정적 상승 | 5~10% 조정 1~2회 |
| **3년차** | 상승 둔화, 박스권 | 10~15% 조정 가능, **현재 위치** |
| 4년차 | 마지막 상승 또는 하락 전환 | 가을→겨울 전환 가능 |

### 5. 연중조정 vs 소공황 구분법
- **연중조정**: -5~10%, 2~3개월 내 회복, 시스템 위기 없음
- **소공황**: -10~20%, 6~12개월, 신용 스프레드 확대 동반
- **2025.09**: 입장 변경 — '연중조정 가능성' → '상승 추세 지속'으로 전환
- **2025.11**: 다시 위험지표 꿈틀 → 금리인하에도 불구하고 리스크 증가
- **핵심 구분**: 비트코인 + 엔비디아가 **동시 하락 전환** 시 → 연중조정 아닌 소공황

## 계절성 패턴과 월별 전략

### 미국 증시 월별 특성
- **1~2월**: 전년 자금 정리, 새해 효과 → 방향성 탐색
- **3~4월**: 세금 환급 시즌 → TGA 변동으로 유동성 영향
- **5~6월**: "Sell in May" → 통계적으로 수익률 낮은 기간
- **7~8월**: 여름 반등 가능 → 특히 대선 사이클과 연동
- **9~10월**: 역사적 변곡점 집중 → 1929, 1987, 2008 모두 가을
- **11~12월**: 산타 랠리 기대 → 연말 포트폴리오 재조정

### 대선 사이클 (4년 주기)
- **대선 직전 해**: 통상 강세 (집권당의 경기부양)
- **대선 해**: 불확실성으로 변동성 확대
- **대선 다음 해(취임 1년차)**: 정책 불확실성 → 조정 가능
- **취임 2년차(중간선거)**: 통상 약세 → 중간선거 후 반등
