# 순환주기와 중기12국면 모델

## 주기 계층 구조

```
장기: ~60년 (콘드라티에프 파동, 인플레/디플레 시대)
  └─ 중기: ~34년 또는 ~17년 (한센 파동, Hansen Cycle)
       └─ 단기: 3~4년 (키친 파동, Kitchin Cycle)
            └─ 초단기: 계절성 (9~11월 변곡점 집중)
```

## 순환주기를 인식하지 못하는 이유
- **중첩된 순환주기**: 중기·단기·초단기 추세가 섞이면 아무런 패턴이 보이지 않음
- 개인투자자가 17년간 주식 투자를 하며 생존하는 경우는 매우 드묾
- 투자 세계가 두려운 이유: 순환주기의 존재를 모르기 때문

## 투자의 매직 넘버 17
- 미국 증시는 약 17년 주기로 대세 상승장과 횡보/하락장이 교대
- 17년을 두 번 반복하여 하나의 34년 순환주기 구성
- 미국 부동산도 17년 순환주기

## 중기4계절 (시즌 개요)

| 계절 | 국면 | 특징 | 투자자 할 일 |
|------|------|------|------------|
| **봄** (3·4·5월) | 비관과 회의의 시대 | 경기 침체 지속, 가장 지루한 국면. 바닥 형성 | 보수적 자산배분, 리밸런싱 (연 2회) |
| **여름** (6·7·8월) | 낙관과 희망의 시대 | 역사적 신고가 돌파, 본격 상승. 엘리엇 상승 3파 | 주도 업종 인화 후 보유 |
| **가을** (9·10·11월) | 도취와 행복의 시대 | 과열 양상, 상승 각도 급증. 금융위기 전조현상 뚜렷 | **가장 바쁜 시기** — 전조현상 면밀 감시 |
| **겨울** (12·1·2월) | 공포와 절망의 시대 | 폭락, 금융위기. 하락 1~3파 | 달러 표시 채권으로 포지션 전환, 매매 중단 |

| 국면 | 엘리엇 파동 대응 | 설명 |
|------|-----------------|------|
| ① | 하락 2파 = 불트랩 | 봄 시작. 반등 후 재하락 가능성 |
| ② | — | 바닥 확인 구간 |
| ③ | 급등 | 상승 1파 시작 |
| ④ | — | 조정 |
| ⑤ | — | 봄 → 여름 전환 |
| ⑥ | 상승 2파 = 베어트랩 | 가장 긴 대형 박스권 |
| ⑦ | 상승 3파 | 여름 본격 상승 |
| ⑧ | 상승 4파 | 잠시 조정 (있을 수도 없을 수도) |
| ⑨ | 상승 5파 | 가을 진입. 상승 각도 급증 |
| ⑩ | — | 가을 과열 |
| ⑪ | — | 하락 1파 시작 |
| ⑫ | — | 겨울 진입 |

## 핵심 투자 타이밍
- **가을 국면(9~11월)**: 전조현상 감지가 가장 중요. 시나리오 투자자에게 가장 바쁜 시기
  - 가을 국면의 핵심 특징 = **상승 각도 급증** (틀립 버블, 닷컴 버블, 일본 버블, 비트코인 등에서 모두 관찰 가능)
  - **버블을 만드는 시간보다 파괴하는 시간이 훨씬 빠르다**: 10년 동안 만든 것도 하루 만에 해체 가능 (복잡 적응계의 티핑 포인트)
- **5월 국면**: 두 번째로 바쁜 시기 — 바닥의 징후 관찰
- **6~8월(여름) → 11월**: 주도 업종을 인화하며 보유
- **겨울 국면**: 달러 표시 채권으로 포지션 전환, 매매 중단
