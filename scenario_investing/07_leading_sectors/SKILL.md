# 주도 업종을 잡아라

## 주도 업종 형성 3가지 요인
1. **시대적·역사적 요구로 발생하는 주도 업종** → 메가 트렌드
   - 한국 증시 1기: 트로이카(troika)의 시대 → 가치주(저PER, 저PBR, 자산주)
   - 한국 증시 2기: 닷컴 → 지주회사 → 차화정 → 바카라
   - 한국 증시 3기: 기술주와 제약바이오 → 선택 소비재

2. **금융 생태계로 자연스럽게 형성된 주도 업종** → 금융 생태계 운명
   - 금융 생태계의 국면에 따라 자연스럽게 부각되는 업종
   - 미국 달러와 한국 증시 주도 업종 비교로 확인

3. **P와 Q가 동시에 성장하는 탁월한 시장의 주도 업종** → 황금 시장
   - 역사적 요구, 금융 생태계와 상관없이 P(가격)×Q(수량) 모두 성장
   - **반도체**가 대표적 예시

## 주도 업종의 세부 조건
1. 업종을 **대표하는 우량 대형주**가 존재하여 전체 업종을 끌고 가야 함
2. 업종 전체가 **어떤 경향성**을 가지고 같이 움직여야 함
3. 실적과 수급 등이 기본 이상이 되어야 하며 **세계적인 시장 점유율**을 가져야 함

## 주도 업종 vs 테마주 — 반드시 구분
| 구분 | 주도 업종 | 테마주 |
|------|---------|--------|
| 대형주 존재 | ✅ 우량 대형주가 견인 | ❌ 소형 부실주 비중 높음 |
| 업종 일관성 | ✅ 업종 전체가 같이 움직임 | ❌ 오합지졸, 업종 대표주 없음 |
| 실적/수급 | ✅ 세계적 점유율 | ❌ 유행성에 의존 |
| 예시 | 반도체, 2차전지 | 정치 테마주, 반짝 유행 |

> 코스닥의 군소 바이오 기업은 임상 결과를 신뢰할 수 없고, 기술·생산 능력이 부족. 삼성바이오/SK바이오 수준이 아니면 테마주로 간주

## 미국 11대 업종과 한국 적용

| 미국 증시 업종 | 한국 적용 시 참고 |
|-------------|----------------|
| Consumer Staple (필수 소비재) | 동일 |
| Consumer Discretionary (선택 소비재) | 동일 |
| Info Tech (기술주) | 미국 통신주 포함 가능 |
| Materials (소재) | 신기술/신시장 소재는 기술주에 가까움 |
| Industries (산업재) | 동일 |
| Utilities (기간산업) | 한국 통신주 포함 (방어주) |
| Financial (금융) | 동일 |
| Energy (에너지) | 신재생 에너지는 기술주에 가까움 |
| Healthcare (헬스케어) | 동일 |
| Communication Services (통신) | 미국은 기술주에 가까움 |
| Real Estate (부동산) | REITs 미발달로 한국에서 제외 |

## 기술주 테크 트리 세분화 예시
```
기술주 카테고리
  └─ 반도체 (대표 업종 선택)
       └─ 대분류: IDM, 파운드리, OSAT, 유통
            └─ 파운드리 중분류: 소재, 부품, 장비, SW
                 └─ 소재 소분류: EUV 소재 등
```
소분류에 가까울수록 구체적인 기술과 유사해진다.

## 기억할 것
- 기술주와 자본재는 모두 경기 민감 업종이지만 완전히 다르다
- **초심자 편향을 버려라** — 과거의 주도 업종에 집착하지 말 것
