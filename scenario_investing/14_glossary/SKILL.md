# 참고 용어집

| 용어 | 설명 |
|------|------|
| 한센 파동 (Hansen Cycle) | 17년 주기 부동산/경기 파동 |
| 키친 파동 (Kitchin Cycle) | 평균 40개월의 단기 경기 파동 |
| 콘드라티에프 파동 | ~60년의 장기 경기 파동 |
| 퍼펙트 스톰 (Perfect Storm) | 두 가지 이상의 악재가 동시 발생해 영향력이 더 커지는 현상 |
| 골디락스 (Goldilocks) | 너무 뜨겁지도 차갑지도 않은 적정한 경제 상황 |
| 그라운드 제로 (Ground Zero) | 금융위기/공황이 폭발한 지점, 분석의 기준점 |
| 니프티 피프티 (Nifty Fifty) | 1970년대 미국 증시를 주도한 우량주 50종목 |
| 제오지산수 | ISM 제조업, 산업생산, OECD선행, GDP, 한국수출 5가지 경기지표 약칭 |
| 불트랩 (Bull Trap) | 하락장에서 일시적 반등으로 매수를 유인하는 함정 |
| 베어트랩 (Bear Trap) | 상승장에서 일시적 하락으로 매도를 유인하는 함정 |
| 창발 (Emergence) | 작은 요소들이 모여 개별에 없는 높은 차원의 특성이 나타나는 현상 |
| 천지인 (天地人) | 천시(시간)·지리(공간)·인화(마음가짐) — 투자 성공의 3대 원칙 |
| 맘느지 투자 | 맘 편한·느린·지속 가능한 투자 (줄임말) |
| PSR (Price-Sales Ratio) | 주가매출비율. 켄 피셔가 주가 급등 정당화에 사용 |
