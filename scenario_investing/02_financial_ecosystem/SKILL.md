# 금융 생태계와 5대 자산

## 5대 자산

| 자산 | 설명 |
|------|------|
| **통화** | 미국 달러(DXY), 원달러 환율(USDKRW) 등 |
| **주식** | 미국 주식, 한국 주식 (성장주/가치주, 대형주/소형주) |
| **채권** | 국채, 회사채 — 금리와 역의 관계 |
| **부동산** | 17년 순환주기(한센 파동) |
| **상품** | 원유, 금 등 원자재 |

- 각 국면별로 자산의 종류, 비율, 등락 속도와 방향이 다르다.
- 5대 자산과 하위 자산들의 특징과 움직임이 **금융 생태계**를 구성한다.
- 5대 자산의 금융 생태계는 **톱니바퀴**처럼 서로 상관관계를 가지고 같이 움직인다. 시계 방향(정방향)이기도, 반시계 방향(역방향)이기도 하다.
- 5대 자산의 톱니바퀴는 모두 직간접적으로 연결되어 있으므로, **미국 달러와 미국 금리** 두 가지만 잘 파악하면 거시 경제의 절반 이상을 이해할 수 있다.

## 4차원 투자 전략
투자 전략을 4가지 축으로 분류:

1. **탑다운 기본(Fundamental)** 전략
2. **탑다운 기술(Technical)** 전략
3. **바텀업 기본(Fundamental)** 전략
4. **바텀업 기술(Technical)** 전략

시나리오 투자법은 주로 **탑다운 기술 전략**에 해당하며, 거시경제 흐름에서 개별 종목으로 내려가는 방식이다.

## 24절기와 시나리오 모델
인류가 변화무쌍한 기후를 24개 국면으로 디지털화하고 국면마다 특징을 덧붙인 것이 24절기다. 24절기가 매우 정확하지는 않지만 **완전히 틀리는 법도 없다**. 이처럼 24절기의 국면별 자연 특성을 자산시장에 접목한 것이 **시나리오 모델**(중기12국면 + 순환주기 + 전조현상)이다.
